## 🛠️ Troubleshooting

If you have trouble opening laws, try these steps:

### App will not open
- Right-click the file and choose Run as administrator
- Make sure the file finished downloading
- Move the app to a simple folder like `C:\laws`

### Windows blocks the file
- Right-click the file
- Open Properties
- If you see an Unblock option, turn it on
- Try opening the app again

### Port is already in use
- Close other local apps
- Change the port in laws
- Start the service again

### App opens but does not connect
- Check that your test app points to the local address
- Make sure the service is still running
- Restart laws and try again

### Data does not save
- Check the data folder path
- Make sure the folder has write access
- Try a new folder in your user profile
